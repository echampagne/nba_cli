# NBA CLI
A command line interface for getting NBA statistics.

## Install

### Build from git source
```bash
git clone git@github.com:echampagne/nba_cli.git
cd nba_cli
python setup.py install
```

## Usage

### Get standings from a conference
```bash
nba --standings --conference=West
nba --standings --conference=East
```

### Help
```bash
nba --help
```
